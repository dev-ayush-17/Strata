# Strata NFT Marketplace - Project Requirements & Screen List ## Brand Overview - **Name:** Strata - **Tagline:** The layer for high-fidelity digital assets. - **Style:** Premium Dark Mode, Modern, Professional. ## Core Screens 1. **Landing Page:** Hero with featured NFT, stats, how it works, and CTA. 2. **Marketplace Explore:** Grid with robust filters, search, and sorting. 3. **NFT Detail:** Focus on the asset, attributes, and purchase/management actions. 4. **Mint NFT:** Multi-step flow for image upload, metadata, and minting. 5. **My NFTs (Profile):** Inventory view for the connected wallet. 6. **Create Listing:** Modal/page for setting price and listing an asset. 7. **My Listings (Dashboard):** Management view for active listings. 8. **Purchase Flow:** Success/Pending/Failure states for buying. 9. **Proceeds Dashboard:** Earnings management and withdrawals. 10. **Wallet Experience:** Connection states and transaction feedback. ## Blockchain Transaction States (Required for all actions) - Awaiting Signature - Transaction Submitted - Transaction Pending - Transaction Confirmed - Transaction Failed ## Design Principles - **Wallet-First:** Clear connection status at all times. - **Trust:** Transparent metadata and transaction feedback. - **Clarity:** Avoid crypto-jargon where possible; prioritize user action.
